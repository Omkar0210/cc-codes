✅ Step 1: Create a Python file

Open terminal and type:

nano vm_kvm.py


Paste your full Python code into the file.

Save and exit:

Press CTRL + O → Enter

Press CTRL + X

✅ Step 2: Verify Python version

Most Ubuntu systems have Python3 installed.

Check version:

python3 --version


If it shows something like Python 3.x.x, you are good.

✅ Step 3: Run the Python script

Use:

python3 vm_kvm.py
